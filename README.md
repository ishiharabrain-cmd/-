# いつでもサポート - 見当識支援アプリ

認知症の方の日常生活をサポートする見当識支援Webアプリケーション（PWA対応）

## 📱 主な機能

- ✅ **時間・日付表示** - 大きな文字で常に表示
- ✅ **現在地確認** - 位置情報を取得して表示
- ✅ **家に帰る** - Googleマップで自宅までのルート案内
- ✅ **動画視聴** - YouTube動画をアプリ内で視聴
- ✅ **写真送信** - 写真を撮影してLINEやメールで家族に送信
- ✅ **服薬リマインダー** - お薬の時間をお知らせ
- ✅ **物探しサポート** - よく使うものの場所を管理
- ✅ **ワンタッチ電話** - 家族やクリニックに簡単に電話

## 🚀 インストール方法

### iPhone/iPadの場合

1. Safariでアプリを開く
2. 画面下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### Androidの場合

1. Chromeでアプリを開く
2. 画面上部の「インストール」をタップ
3. または、メニュー（⋮）から「ホーム画面に追加」を選択

### パソコンの場合

1. Chrome/Edgeでアプリを開く
2. アドレスバー右側の「インストール」アイコンをクリック
3. 「インストール」をクリック

## ⚙️ 初期設定

アプリを初めて起動すると、設定画面が表示されます。以下の情報を登録してください：

1. **基本情報**
   - お名前
   - 自宅住所

2. **家族の連絡先**
   - 家族の名前と電話番号（2名まで）
   - 写真送信先のメールアドレス

3. **クリニック情報**
   - クリニック名
   - 電話番号

4. **服薬時間**
   - 朝・昼・夕方・夜の各時間帯

5. **写真送信方法**
   - LINE/メール/毎回選択

## 💡 使い方のヒント

### 位置情報の許可
初回起動時に位置情報の使用許可を求められたら「許可」を選択してください。現在地の表示と「家に帰る」機能に必要です。

### カメラの許可
写真を撮影する際、カメラの使用許可を求められたら「許可」を選択してください。

### ホーム画面への追加
ホーム画面に追加することで、通常のアプリのように簡単に起動できます。

### オフライン使用
一度アプリを読み込めば、インターネット接続がなくても基本機能が使えます。

## 🔧 技術仕様

- **フロントエンド**: HTML5, CSS3, JavaScript (Vanilla)
- **PWA対応**: Service Worker, Web App Manifest
- **API使用**: 
  - Geolocation API（位置情報）
  - Camera API（写真撮影）
  - YouTube Embed API（動画再生）
  - Google Maps API（ナビゲーション）

## 📝 カスタマイズ

### 服薬時間の変更
設定画面（⚙️）から各時間帯の服薬時間をカスタマイズできます。

### 家族連絡先の追加
現在は2名まで登録可能です。さらに追加したい場合は設定画面のコードを編集してください。

### デザインのカスタマイズ
各HTMLファイルの`<style>`タグ内でボタンの色やサイズを変更できます。

## 🔒 プライバシーとセキュリティ

- すべての設定情報はデバイスのローカルストレージに保存されます
- サーバーへのデータ送信は行いません
- 位置情報は現在地の表示とナビゲーションにのみ使用されます
