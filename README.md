# 伴走者アプリ - やさしい音声操作アプリ

**認知症の方・高齢者・すべての人のための音声ファーストアプリ**

---

## 特徴

### 🎤 音声で全操作
- マイクボタンを押して話すだけ
- 運転中・料理中でもハンズフリー操作
- 数字（1〜9）でも操作可能

### 📱 誰でもすぐ使える
- Googleアカウント不要
- 設定ゼロで即使用可能
- データはスマホ内に安全保管

### 🔤 読みやすい
- Fire TV級の大きな文字
- ひらがな/漢字切替
- 高コントラスト

### 🌍 完全無料
- 基本機能は全て無料
- AIチャット機能のみAPI登録が必要

---

## 機能一覧

### 基本機能（無料）
- 📅 予定管理（カレンダー）
- ✅ Todo管理
- 📝 記録（ライフログ）
- 📞 家族に電話
- 🗺️ 地図・道案内
- 📺 動画（暇つぶし）
- 🆔 あいさつ（迷子札）

### AI機能（APIキー登録が必要）
- 🤔 壁打ち・相談相手
- 💬 雑談・心のサポート
- 🧠 認知症の相談

## 使い方

### 1. 音声操作
マイクボタン🎤を押して話す：
- 「家に帰る」
- 「予定を追加して：明日の午後2時に病院」
- 「Todoに追加して：牛乳を買う」
- 「記録して：今日は公園に行きました」

### 2. 数字キー操作
キーボードまたは音声で数字を入力：
- **2** - 記録する
- **3** - 動画
- **4** - 電話
- **6** - おしゃべり
- **8** - 場所
- **9** - あいさつ

### 3. 画面タップ操作
- 天気をタップ → 天気予報詳細
- 日付をタップ → カレンダー
- ➕ボタン → 予定/Todo追加

## APIキーの設定（AIチャット機能を使う場合）

### 手順
1. 右上の **⚙️設定** をタップ
2. **🔐 Gemini APIキー** セクション
3. 「➕ APIキーを設定」をタップ
4. [Google AI Studio](https://aistudio.google.com/app/apikey) でAPIキーを取得
5. 入力して保存

### 料金
- Gemini 2.0 Flash: 入力$0.10/100万トークン、出力$0.40/100万トークン
- 1回のチャット: 約0.03円
- 1日10回使用: 約0.3円
